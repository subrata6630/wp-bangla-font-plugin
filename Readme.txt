== Description ==
Description:  An easy way to add Bangla fonts to any theme without coding using WordPress Customizer




== Installation ==

Just Upload `Bangla Font`  to the `/wp-content/plugins/` directory
Activate the plugin through the 'Plugins' menu in WordPress

<style>
  body {
    font-family: 'Hind Siliguri', sans-serif !important;
  }
</style>

And enjoy this plugin!
